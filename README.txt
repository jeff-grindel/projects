Project: Arduino Sous Vide
Author: Jeff Grindel

Description: A program that utilizes a Arduino Microcontroller to implment a water bath cooking
style. 

Setup: Run and compile .ino file utilizing the Arduino sowtware package. Upload the code to the 
Arduino and bam you are ready to cook. Need to ensure that all the wiring is setup. Wiring setup
can be found at http://jeffgrindel.com/SousVide.html
